# Changelog

## 0.1.1

- One universal macOS DMG (Apple Silicon + Intel) replaces the zip.
- Homebrew install: `brew install --cask 0xharkirat/tap/downpour`.
- Format choice: MP4 (plays everywhere, default) or best quality (WebM/MKV).
- Links dragged from the browser address bar now work.
- Merged downloads label their second pass as the audio track.

## 0.1.0

First public release.

- Paste, drop, or type a video link to see its details (thumbnail, title, uploader, length), pick a quality, then download.
- Works with every site yt-dlp supports: YouTube, Vimeo, Dailymotion, X, and 1800+ more.
- Zero setup: yt-dlp and ffmpeg are found on your system or downloaded automatically on first launch.
- Quality presets (Best, 4K, 1080p, 720p, audio-only MP3) with the actually downloaded resolution shown when done.
- Live progress with speed and ETA, cancel, open, reveal in folder, and download again.
- Download history persists across restarts, split from in-progress items.
- Video transcripts from platform captions, with a text sidecar saved next to the video.
- System notification when a download finishes; click it to open the file.
- Engine updater: check for new yt-dlp releases and install in one click.
- Light, dark, and system themes.
